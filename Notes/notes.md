#### Fontes
    - Conversor de fontes: TTF para Woff2 https://cloudconvert.com/ttf-to-woff2

    1. Baixa a fonte
    2. Converta ela para o formato woff2
    3. Insira a fonte no CSS:
        @font-face {
            font-family: NomeFont, tipoFont;
            src: url(fonteX.ttf);
            src: url(fonteY.woff2); //Fonte alternativa
        }
